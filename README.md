# Playable Piano - Controller Support

## 📖 About
This is a **controller support update** for Mushroomy's "Playable Piano" mod. It adds the ability to control the MIDI playback menu using a game controller.

## 🎮 What was changed
- **Controller support for MIDI playback menu**
  - **X / Y (Xbox) / Square (PlayStation)** → Open piano menu
  - **B (Xbox) / Circle (PlayStation)** → Close menu / Cancel
  - **Escape / Right Click** → Same functions (unchanged)

## ❌ What was NOT changed
- Free Play mode (keyboard only)
- Click detection
- Playback controls
- Transitions

## 🎹 Original Features (Still work the same)
- Play Dark Piano and Upright Piano
- MIDI file playback
- Free play mode (keyboard)

## 📥 Installation
1. Install **SMAPI**
2. Download and unzip into `Stardew Valley/Mods/`
